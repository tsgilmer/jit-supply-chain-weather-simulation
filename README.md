# jit-supply-chain-weather-simulation

JIT Supply Chain Weather Resilience Simulation

This project models weather-driven supply chain disruptions in a dual-line automotive assembly plant using Monte Carlo simulation.

The model evaluates:

JIT truck delivery logistics

inventory buffer resilience

seasonal weather disruptions

supply corridor vulnerabilities

Simulation tools:

Python

NumPy

Matplotlib

Google Colab

The results identify a critical inventory threshold (~360 parts) required for the plant to survive logistics disruptions.
